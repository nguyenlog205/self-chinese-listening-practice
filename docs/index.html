Học theo cách phù hợp với bạn

Một ứng dụng, nhiều hình thức luyện tập — từ ghi nhớ từ vựng tới phản xạ nghe câu dài.

词

Từ vựng HSK 1 → 7-9

Flashcard, nghe‑gõ chính tả, trắc nghiệm, xếp chữ đúng thứ tự và cả đề thi thử — theo từng cấp độ.

话

Luyện hội thoại

Nghe đoạn hội thoại thật, trả lời trắc nghiệm, điền vào chỗ trống, hoặc gõ lại nguyên câu (Dictation).

▶

Luyện nghe qua YouTube

Dán một link video tiếng Trung — hệ thống tự tách câu, chuyển pinyin, cho bạn luyện nghe từng câu một.

🔥

Theo dõi tiến trình

Chuỗi ngày học liên tiếp, biểu đồ hoạt động, tiến độ theo từng cấp HSK — lưu hoàn toàn local.

文

Giản thể ⇄ Phồn thể

Chuyển đổi chữ Hán và cả Pinyin ⇄ Chú âm (Zhuyin) bất cứ lúc nào, không ảnh hưởng dữ liệu gốc.

语

4 ngôn ngữ giao diện

Tiếng Việt, English, 简体中文, 繁體中文 — đổi ngay trong Cài đặt, áp dụng cho toàn bộ ứng dụng.

Gọn gàng, tập trung vào việc học

Không quảng cáo, không thông báo làm phiền. Giao diện chỉ hiện những gì bạn cần cho phiên luyện tập hiện tại — pinyin, phiên âm, bản dịch đều có thể bật/tắt theo ý bạn.

  • Cập nhật từ vựng & hội thoại mới nhất chỉ với một cú nhấp
  • Phát âm chuẩn bằng giọng đọc tự nhiên (edge-tts)
  • Toàn bộ tiến trình lưu trên máy — xoá app không mất gì bạn không muốn mất

Tải xuống

Chọn hệ điều hành của bạn. Ứng dụng chạy offline, không cần tài khoản.

🐧 Linux (Fedora)

Cài qua gói .rpm chuẩn, tích hợp menu ứng dụng và dnf.

sudo dnf install -y python3 ffmpeg
sudo dnf install -y ./listening-electron-*.rpm
listening-electron

Không dùng Fedora? Bản .AppImage chạy được trên mọi distro Linux — chỉ cần chmod +x rồi mở.

🪟 Windows

Chạy trực tiếp file cài đặt .exe — không cần quyền quản trị.

  1. Tải file ListeningPractice Setup.exe
  2. Double-click để chạy trình cài đặt
  3. Mở "ListeningPractice" từ Start Menu

Cần cài sẵn Python 3.10+ và ffmpeg trong PATH.

Mã nguồn & hướng dẫn build: github.com/nguyenlog205/self-chinese-listening-practice
