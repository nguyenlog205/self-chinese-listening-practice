# Benchmarking Datasets

Thư mục này chứa các datasets sử dụng để benchmarking các mô hình STT. Mỗi dataset được tổ chức trong một folder riêng biệt với README chi tiết.

## Cấu trúc

```
data/
├── README.md                          # File này
├── aishell-1/                         # Clean, controlled speech
├── aishell-2/                         # Diverse speakers
├── common-voice-zh/                   # Crowd-sourced Chinese
├── synthetic-noisy/                   # Augmented with noise
└── youtube-samples/                   # Real-world content
```

Mỗi folder chứa:
- README.md: Mô tả chi tiết dataset
- audio/: Thư mục chứa audio files
- transcriptions.json: Ground truth transcriptions
- metadata.json: Metadata về dataset
